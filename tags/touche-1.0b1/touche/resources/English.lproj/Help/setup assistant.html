Setup assistant

The setup assistant is intended to faciliate the setup of your Touché installation to get you up and running quickly. You can either start it from the welcome screen the first time you launch Touché or by clicking the "Setup Assistant..." item in the "Setup" menu.

For each step, there will be a short description of what to do as well as buttons to open the appropriate configuration dialogs. Note that the "Continue" button will be greyed out until you have finished the necessary action for this step.

It is recommended to run the setup assistant at least once, in order to make sure that you haven't forgotten an important configuration item.
