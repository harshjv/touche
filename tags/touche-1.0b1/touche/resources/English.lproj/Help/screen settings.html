Screen settings

You can access Touché's screen settings configuration dialog by clicking the "Screen setup..." item in the "Setup" menu.

By configuring the screen, you can make sure which screen connected to your Mac refers to the projector you use for your FTIR multitouch table, but also to ensure proper sizing of screen elements independent of the projected area's absolute size.

Screen to use for multitouch applications

Select the screen you want to use for your multitouch applications. Usually, this should be the screen that refers to the projector used in your FTIR setup. Note that Touché will fall back to your Mac's main screen if the screen chosen here is not active when you launch Touché.


Screen resolution

Due to the fact that your FTIR table is (most likely) using a rear-projection mechanism, the resolution of your multitouch screen area in terms of "pixels per inch" or "pixels per centimeter" is not known. Therefore, graphical elements and widgets might differ vastly in their absolute size when used on different FTIR tables.

In order to prevent this effect, you can set the "pixels per inch" resolution of your FTIR table's screen area here: Touché will then use your setting to size its multitouch elements accordingly. Also, the Touché framework provides a way for third-party multitouch apps to read out the amount of pixels per centimeter or inch, so that they can size their controls accordingly as well.

To set the resolution, simply drag the screen configuration panel to the screen of your FTIR table and then measure the length of the grey line with a ruler. Then, enter the measured size via the slider.
