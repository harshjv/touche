Preferences

In Touché's preferences, you can set some miscellaneous user settings. Note that the tracking-specific features are accessible in the "Setup" menu instead.

Interface color

The interface color refers mainly to the color of the status bar in the main window. You can choose any color that might suit you better than default. If you want to revert your choice, hit the "Reset" button. This is a purely aesthetic feature :-)


Animate touch indicators

By checking this checkbox, you can have the touch indicators shown in the calibration and test application animate in a pulse motion, rather than being static disks. This is also a purely aesthetic feature, but be aware that the touch animation might slow down your Mac a lot, depending on the graphics card.


Check for updates on startup

If this checkbox is set, Touché will connect to the internet and look for updates to itself. If an update is found, you can install it easily right from within the application. No personal information is transmitted during this process.

Since updates will often introduce new feature or fix bugs, it is heavily recommended to have this option enabled.
