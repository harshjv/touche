Configuring Touché

Touché's configuration can be split into four different parts: Screen setup, tracking pipeline setup, calibration and miscellaneous preferences. The first three can be accessed via the "Setup" menu, whereas the miscellaneous preferences are accessible via the main menu by selecting the "Preferences..." menu item.
