Blob input preview window

The blob input preview window shows a live stream of the video frames the current input source is capturing. You can open this window either by selecting "Blob Input Preview Window" in the "Window" menu or by clicking on the "Larger Preview" button in the blob input method settings box.

If the currently active input method supports filtering, you can choose the filter stage from which the preview video stream should be taken. This is a very useful feature for diagnosing problems with your optical multitouch table. Note that if the currently active input method does not support filtering, this menu will be disabled.
