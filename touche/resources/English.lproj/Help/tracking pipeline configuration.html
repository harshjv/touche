Tracking pipeline configuration

You can open the tracking pipeline configuration window by clicking the "Configure Tracking Pipeline..." menu item in the "Setup" menu.

Depending on the selected input source, the configuration window will look differently. However, the top portion of the window is always present. This is the blob input method configuration box.

For further information about the configuration of the different pipeline elements, refer to the "related topics" list on this page.
