Telling multitouch applications to quit

You can tell client applications to quit by right-clicking (or control-clicking) an entry in the list and selecting "Tell client ot quit" in the contextual menu.

Quitting client applications this way can be useful if they do not have their own mechanism to quit them. Note that this method will only send a message to the client application, but won't tear it down directly. The message might be ignored by a client on purpose, so that it may not quit as instructed.
