Supported input devices

Touché currently supports the following input methods:

  • QuickTime-compatible camera
  • libdc1394-compatible FireWire camera
  • Wii Remote

You can select your preferred input method in the blob input method settings box.

Note that when using a Wii Remote, your Mac must support Bluetooth, and that the Wii Remote can only track up to 4 touch points at once.

Note that for an optical multitouch table, you will need to use an infrared camera. However, you can easily mod many webcams to infrared cameras by replacing the the "visible light" bandpass filter within them with an infrared filter (for which you could use scrap parts of a developed photo film, for example). Please search the web for more information on how to mod a camera and build a multitouch table.
