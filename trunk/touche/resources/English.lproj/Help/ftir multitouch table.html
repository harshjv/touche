FTIR multitouch table

FTIR stands for "frustrated total internal reflection" and is an effect of evanescent waves used in many applications such as fingerprint scanners and optical fibres.

A simple description of this effect is that light (and infrared) waves injected into a sheet of acrylic glass through the sides will "bounce" within this light until it is disturbed by an object with an appropriate refractive index touching the glass. This effect can thusly be used to track multiple touches on a piece of acrylic glass by first injecting infrared light into the glass, then capturing the IR light refracted by a finger with an infrared camera. Software can now be used to track these touch points over multiple frames, compute the position of a touch in screen coordinates, etc.

Touché is such a software.

Additional information on the physics behind FTIR can be found on Wikipedia.
