Getting started with Touché

When you launch the Touché application for the first time, it will offer to run an assistant to guide you through the setup process. You can also re-run this assistant at any time by clicking the "Setup Assistant..." menu item in the "Setup" menu.

Be sure to have your optical multitouch table and infrared camera ready before running the setup assistant.
