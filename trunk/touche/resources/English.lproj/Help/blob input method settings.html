Blob input method settings

Currently, Touché supports 3 different input methods: QuickTime-compatible cameras, libdc1394-compatible cameras and Wii Remotes (if your Mac supports Bluetooth).

You can select the input method most suitable for you via the drop-down menu in the tracking pipeline configuration view. Depending on the method selected, different options will be available below the input method selection box.


Show detected blobs in preview

If this box is checked, detected blobs will be shown as colored discs in the preview area as well as in the large preview window. Being able to switch this on and off is useful during the configuration of the pipeline.


Filter stage for preview video stream

If the selected input method supports image filtering, you can select the filter stage from which the video stream in the preview area and preview window is taken. This is very useful during the configuration of the various filters. If the input method does not support filters, this menu will be disabled.


Larger preview

Clicking this button opens the blob input preview window.
