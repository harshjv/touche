Introduction

Touché is a touch tracking application for use in conjunction with an optical multitouch table. It provides powerful image filters, built-in calibration and test tools as well as support for types of cameras as input input sources.

Additionally, Touché enables you to build your own multitouch applications by embedding the Touché framework in your own app. Via this framework, you can establish a connection to the tracker and receive data for touch events conveniently via delegate methods. The framework also includes some useful classes to facilitate the writing of multitouch applications, such as some gesture recognizers.

Touché also supports the TUIO protocol, which you can also use as a means to relay tracking data to your own application. TUIO libraries are available for a large range of environments and languages, such as C++, Processing and Quartz Composer.

Additionally to TUIO/OSC, Touché can also output tracking data via TUIO/Flash XML-Socket, which is convenient if you are using Flash/AS3 to develop multitouch applications.

Touché is free open-source software.
