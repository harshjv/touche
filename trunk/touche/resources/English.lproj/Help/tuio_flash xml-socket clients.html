TUIO/Flash XML-Socket clients

Flash/AS3 is a popular platform to develop multitouch applications with. The touchlib project includes a parser for the TUIO protocol, which is translated from OSC to Flash XML-Sockets using the open-source tool flosc.

However, Touché can natively output tracking data in the same format that flosc translates TUIO data into, allowing you to use Flash clients without needing to use flosc.

When starting a Flash application, it can connect to the XML-Socket server provided by Touché, which will then deliver tracking data to the application. Per default, the XML-Socket server listens on port 3000, but this can be changed in the TUIO/Flash XML-Socket settings.

XML-Socket clients show up in the tracking client list just like proper Touché framework clients do.
