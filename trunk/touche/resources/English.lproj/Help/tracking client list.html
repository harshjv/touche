Tracking client list

If one or more multitouch applications are connected to the Touché tracking server, the main window will display a list of these client application. Each entry includes the name, the icon and the version number of the multitouch application.

By right-clicking (or control-clicking, if you don't have a right mouse button) an entry in the list, you can bring up a contextual menu.

Note that if no clients are currently connected, the list will not be shown. A message saying "No clients connected" will be displayed instead.
